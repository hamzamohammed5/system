# ERP BOM System

## تشغيل البرنامج
```bash
cd erp
python main.py
```

## هيكل الملفات

```
erp/
├── main.py              ← نقطة الدخول (شغّل هذا)
├── database.py          ← كل عمليات SQLite في مكان واحد
├── logic.py             ← حسابات التكلفة (منفصلة عن UI و DB)
├── app.py               ← النافذة الرئيسية (تجمع التبويبات فقط)
└── tabs/
    ├── __init__.py
    ├── raw_tab.py        ← تبويب الخامات (مكتفي بنفسه)
    ├── product_tab.py    ← تبويب المنتجات semi/final (مكتفي بنفسه)
    └── component_row.py  ← ويدجت صف مكون (قابل لإعادة الاستخدام)
```

## مبادئ التصميم

| المبدأ | التطبيق |
|--------|---------|
| **فصل المسؤوليات** | DB / Logic / UI كل واحد في طبقة مستقلة |
| **تبويبات مكتفية** | كل تبويب له `connection` خاص، لا يستدعي كودًا من تبويب آخر |
| **ويدجت قابل لإعادة الاستخدام** | `ComponentRow` يُستخدم في نموذج الإنشاء |
| **أزرار في مكانها** | أزرار الإنشاء بجوار نموذج الإنشاء، أزرار الشجرة تحت الشجرة |

## كيف تضيف ميزة جديدة؟

- **حقل جديد في items** → عدّل `database.py` فقط (أضف العمود في `CREATE TABLE` وفي الـ functions)
- **حساب مختلف للتكلفة** → عدّل `logic.py` فقط
- **تبويب جديد** → أنشئ ملف في `tabs/` واستورده في `app.py`
- **تغيير DB** → عدّل `database.py` فقط، باقي الكود يبقى كما هو
