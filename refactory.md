**قواعد الإخراج — لا استثناء:**

## قاعدة صارمة قبل كتابة أي ملف

قبل إخراج أي ملف معدَّل، نفّذ الخطوات دي بالترتيب:

١. اقتبس أول 3 أسطر غير فارغة من الملف المرفوع كما هي (في تعليق `#` داخل الكود).
٢. طبّق التعديلات المطلوبة فقط على باقي المحتوى — لا تعيد ترتيب الكود، لا تحذف أي سطر غير مستهدف بالفحص.
٣. ممنوع حذف أي import أو دالة أو class موجود في الأصل إلا إذا كان مستهدفاً صراحةً بالفحص (ب).

**اختبار ذاتي:** إذا كان الملف الناتج يحتوي على بنية أو imports أو دوال غير موجودة في الملف المرفوع الأصلي → احذفها.

## الفحص (أ): القيم الـ Hardcoded

### ما يُعدّ hardcoded (داخل كود قابل للتنفيذ فقط، لا docstring أو تعليق)

**١. لون مباشر** — أي لون لا يأتي من `_C[...]`
**٢. حجم خط مباشر** — أي رقم لا يأتي من `FS_*` أو `get_font_size()` أو `fs()` من `ui/font.py`
**٣. نص معروض للمستخدم غير مُترجَم** — أي نص يظهر في الواجهة ولا يمر عبر `tr()`، يشمل:
- نصوص الأزرار، labels، tooltips، placeholders، addTab، addItem
- رسائل QMessageBox
- f-strings فيها نص ثابت معروض
- class-level attributes تُستخدم كنصوص في الواجهة
- فواصل بصرية مثل `"──"`, `"━"`, إيموجي معروض مثل `"📦"`, رموز unicode مثل `"→"`

**خارج الفحص:** docstrings، تعليقات `#`، أسماء متغيرات/دوال، strings تُستخدم كمعرّفات برمجية داخلية.

### وسائل حجم الخط المعتمدة

اقرأ الملف المرفوع `ui/font.py` مباشرةً لتحديد ثوابت `FS_*` الموجودة وقيمها — لا تعتمد على أي قيم محفوظة في ذاكرتك.

- **ثابت** → استبدل بأقرب `FS_*`
- **ديناميكي** → `get_font_size()` داخل `_refresh_style()`
- **نسبي** → `fs(get_font_size(), delta)` أو `fs(FS_BASE, delta)`
- لا تضف مفاتيح جديدة لـ `font.py`.

### الإجراء

**إذا كان الملف المرجعي مرفوعاً:** عدّل مباشرة:
1. ابحث عن مفتاح مطابق بالمعنى في الملف المرجعي.
   - للألوان: ابحث في **كل** modes في `theme_manager.py`، وأي مفتاح جديد يُضاف يجب تعريفه في كل mode.
   - للنصوص: المفتاح يجب أن يكون بنفس الاسم في **كلا** `ar.py` و`en.py` معاً.
2. إذا وجدت مفتاحاً → استخدمه.
3. إذا لم تجد → أنشئ مفتاحاً جديداً بنفس naming convention الملف، أضفه للملف المرجعي، واستبدل القيمة في الـ widget.
4. تحقق من صحة الـ syntax بعد التعديل.

**إذا كان الملف المرجعي غير مرفوع:** أدرج القيمة في قسم "بانتظار ملفات مرجعية ناقصة" فقط.

## الفحص (ج): توحيد الثوابت عبر `ui/constants.py`

### ما يخضع للفحص
كل قيمة رقمية صلبة داخل كود قابل للتنفيذ من هذه الأنواع:
- **أبعاد وأحجام** — عرض، ارتفاع، هامش، padding، spacing، border-radius
- **حدود رقمية** — min/max لحجم خط، عدد عناصر، حدود تحقق
- **قيم افتراضية هيكلية** — أبعاد نافذة، أبعاد sidebar، timeout، عدد أعمدة

**خارج الفحص:** ألوان، أحجام خط، نصوص مترجمة، معرّفات برمجية داخلية.

### الثوابت المعرَّفة حالياً في `constants.py`

اقرأ الملف المرفوع `constants.py` مباشرةً لتحديد الثوابت الموجودة — لا تعتمد على أي قيم محفوظة في ذاكرتك.

### الإجراء

**إذا كانت القيمة موجودة:** استبدلها بالثابت مع إضافة الـ import.

**إذا لم تكن موجودة:** أضفها لـ `constants.py` بـ SCREAMING_SNAKE_CASE تحت القسم المناسب، ثم استخدمها في الـ widget.

## الفحص (ب): تحويل الـ widgets لـ WidgetMixin

### نطاق التحويل
كل widget يستخدم مباشرةً **داخل تعريفه** أياً من: `_C`، `get_font_size()/fs()/FS_*`، `tr()`، أو bus events.

**خارج النطاق:** استخدام غير مباشر — لا تتبع الاستدعاء ولا تعدّل الكلاس الآخر.

### خطوات التحويل

**١. تعديل تعريف الـ class**
```python
from ui.widgets.core.widget_mixin import WidgetMixin
class MyWidget(QComboBox, WidgetMixin):
```

**٢. في `__init__`**
```python
self._init_widget_mixin()   # الافتراضي: theme=True, font=True, lang=True, data=True
self._refresh_style()
```

مرّر `False` فقط لما لن تحتاجه:

| `False` مناسب لـ | متى |
|-----------------|-----|
| `lang=False` | لا نصوص مترجمة |
| `data=False` | لا بيانات من DB |
| `font=False` | لا نص معروض |
| `theme=False` | لا ألوان من `_C` (نادر) |

> `data=True` يستدعي `company_state.company_id` عند الـ init — مرّر `data=False` إذا كان الـ widget لا يستخدم DB.

**٣. تعريف الدوال**
```python
def _refresh_style(self, *_):
    from ui.theme import _C
    from ui.font import get_font_size
    self.setStyleSheet(f"color:{_C['text_primary']}; font-size:{get_font_size()}px;")

def _refresh_lang(self, *_):      # فقط إذا lang=True
    from ui.widgets.core.i18n import tr
    self.setPlaceholderText(tr('search'))

def _refresh_data(self, company_id=None):   # فقط إذا data=True
    ...
```

**٤. احذف**
- `weakref.ref(self)` المرتبط بالـ bus
- `bus.X.connect(...)` اليدوي
- استدعاء `_C`/`get_font_size()`/`tr()` مرة واحدة في `__init__` — يُنقل لـ `_refresh_style`/`_refresh_lang`
- imports غير مستخدمة بعد التحويل
- `closeEvent` يفصل slots يدوياً
